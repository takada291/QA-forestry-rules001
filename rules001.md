# 伐採造林届出書作成の手引き（令和６年１月版）

本手引きは、森林法（昭和26年法律第249号）第10条の８・第10条の９に基づく「伐採及び伐採後の造林の届出書」（以下、「伐採造林届出書」）を作成する森林所有者や事業者向けに作成方法を解説したものです。

## １ 伐採造林届出制度の概要

### ① 制度の概要
* 森林法に基づき、立木の伐採を行う場合に、市町村長へ「伐採造林届出書」の提出が義務付けられています。
* 市町村は、届出内容が「市町村森林整備計画」に適合しているか確認し、必要に応じて指導を行います。
* 無届伐採や計画と異なる施業を行った場合、伐採中止や計画変更、造林命令、罰則の対象となる場合があります。

### ② 制度の主な流れ
1. **届出書の提出（伐採の90〜30日前）**：森林所有者等が市町村へ提出。
2. **内容確認・通知**：市町村が適合（確認）通知を発行、または指導。
3. **伐採の開始〜終了**
4. **伐採報告の提出（伐採後30日以内）**：森林所有者等が市町村へ提出。
5. **造林の開始〜終了**
6. **造林報告の提出（造林後30日以内）**：森林所有者等が市町村へ提出。

### ③ 届出対象の森林
* 森林法第5条に基づき、都道府県知事がたてる「地域森林計画対象の民有林」です。

### ④ 届出を行う者
* 森林法第10条の7に規定される「森林所有者等」（立木を伐採する者）。
* 森林所有者のほか、立木を購入した者、森林経営委託を受けた者が含まれます。
* **ポイント**：伐採者と造林者が異なる場合は、連名で届け出る必要があります。

### ⑤ 届出を行う時期
* 伐採を行う**90日から30日前まで**。

### ⑥ 提出が不要となる場合（適用除外）
* 保安林での伐採（別途、保安林関係の手続きが必要）。
* 林地開発許可を受けた場合。
* 市町村の特定間伐等促進計画に従う場合。
* 森林経営計画に基づく伐採（事後届出が必要）。
* 災害時の緊急伐採、除伐、枯死木の伐採など。
* **ポイント**：竹の伐採や、一般的な枝払いは届出不要です。

### ⑦ 届出対象のフローチャート
1. **保安林等に指定されているか？**
    * はい → 保安林関係手続きが必要。
    * いいえ → 次へ。
2. **地域森林計画の対象民有林か？**
    * いいえ → 届出不要。
    * はい → 次へ。
3. **1ha超（太陽光は0.5ha超）の土地の形質変更に該当するか？**
    * はい → 林地開発許可が必要。
    * いいえ → 次へ。
4. **届出の適用除外に該当するか？**
    * はい → 届出不要。
    * いいえ → **伐採造林届出制度の対象**。

## ２－１ 伐採造林届出書の内容・記載例

### ① 届出書（表紙）の記載項目
* **提出先**：森林のある市町村長。
* **届出人**：伐採する者、造林する者の住所・氏名（押印不要）。
* **所有者**：伐採する立木の所有者。
* **森林の所在場所**：市町村、大字、字、地番。
* **備考**：森林法以外の法令制限や、適合通知書の希望の有無。

### ② 伐採計画書
* **伐採面積**：小数点第2位まで記載。人工林・天然林の内訳。
* **伐採方法**：主伐（皆伐・択伐）、間伐。
* **伐採率**：
    * 皆伐：100%
    * 択伐（人工造林）：40%上限
    * 択伐（天然更新）：30%上限
    * 間伐：35%上限（※市町村により異なる場合あり）
* **伐採樹種・伐採齢**：森林簿等を参考に記載。
* **集材方法**：集材路、架線等。集材路の幅員・延長も記載。

### ③ 造林計画書
* **造林面積**：伐採計画の主伐面積と一致させる。
* **造林の方法**：人工造林（植栽・人工播種）、天然更新（ぼう芽更新・天然下種更新）。
* **造林の期間**：
    * 人工造林（皆伐）：2年以内
    * 天然更新：5年以内
* **樹種別の植栽本数**：市町村森林整備計画に定める基準に従う。
* **鳥獣害対策**：防護柵の設置、幼齢木保護具の設置など。

## ２－２ 添付書類

| 番号 | 項目 | 備考 |
| :--- | :--- | :--- |
| ① | **位置図・区域図** | 必須。森林の位置が特定できる図面。縮尺任意。 |
| ② | **届出者の確認書類** | 必須。法人は登記事項証明書、個人は住民票や免許証の写し等。 |
| ③ | **許認可関係書類** | 該当する場合のみ。他の行政庁の許可書の写し等。 |
| ④ | **土地権原関係書類** | 必須。土地の登記事項証明書、売買契約書等。 |
| ⑤ | **伐採権原関係書類** | 届出者が土地所有者でない場合。立木売買契約書等。 |
| ⑥ | **境界確認関係書類** | 必須（省略可能要件あり）。隣接所有者との境界確認状況。 |
| ⑦ | **その他** | 市町村長が必要と認めるもの（搬出計画図、チェックリスト等）。 |

### 森林作業道・集材路の作設留意点
* **要点**：
    * 縦断勾配は10度（18%）以下を基本とする。
    * 幅員は必要最小限（2.5m〜3.0m）とする。
    * 切土高は1.5m以内が望ましい。
    * 小まめな分散排水（波状勾配、横断溝）を行う。

## ４ 伐採・造林の状況報告

### ① 伐採後の状況報告書
* **提出期限**：伐採（主伐に限る）が終了してから**30日以内**。
* **内容**：実施した伐採面積、方法、樹種、集材方法などの実績。

### ② 造林後の状況報告書
* **提出期限**：造林が終了してから**30日以内**。
* **内容**：実施した造林方法、期間、樹種、面積、本数などの実績。

### ③ チェックポイント（報告時）
* 報告日が期限内か。
* 所在場所が届出書と一致しているか。
* 面積や方法が提出した計画と整合しているか。
* 市町村から指導されている書類（写真等）が添付されているか。
